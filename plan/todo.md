# Что нужно сделать

## 🔴 Критично (сделать первыми)

### 1. Подключить бота через BotFather
- Открыть @BotFather → `/newapp` или `/myapps`
- Указать ссылку: `https://my-mini-app-ebon.vercel.app`
- После этого Mini App открывается прямо из Telegram

### 2. Вставить ссылки оплаты ← ждём от пользователя
- В `tg-app/js/data.js` заменить `payUrl: ''` на реальные ссылки для каждого ассистента
- После подключения кнопка автоматически сменится на `"Купить за N ₽"`

---

## 🟡 Важно (следующая итерация)

### 3. Онбординг для новых пользователей
- Один экран при первом запуске: "Что такое AI-ассистент и как это работает"
- Показывать только один раз (через `localStorage`)

### 4. Элемент срочности
- "Цена действует до [дата]" или "Осталось N активаций" на карточке
- Добавить поле `urgency` в данные ассистента в `data.js`

### 5. Бесплатный демо-ассистент
- Один ассистент с `price: 0` как точка входа
- Убирает барьер первой покупки

---

## 🟢 Улучшения (по возможности)

### 6. Тёмная тема через tg.themeParams
- Автоподхват цветов через `tg.themeParams.bg_color`, `tg.themeParams.text_color`
- Идеальная нативность с темой пользователя

### 7. Success screen — реальная интеграция с оплатой
- Сейчас Success показывается сразу при нажатии "Купить" (демо-режим)
- В production нужен webhook от платёжного сервиса → бэкенд → redirect с `?success=1&id=X`

## Порядок работы
1. Подключить бота (BotFather) — чтобы Mini App открывался из Telegram
2. Вставить `payUrl` когда будут ссылки оплаты
3. Онбординг (п.3) — для новых пользователей
4. Срочность (п.4) — поднимает конверсию
